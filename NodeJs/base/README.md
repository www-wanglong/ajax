## 异步IO
- IO是应用程序的瓶颈所在
- 异步IO提高性能无菜原地等待结果返回
- IO操作属于操作系统级别，平台都有对应的实现
- NodeJs单线程配合事件驱动架构及libuv实现了异步IO

## Buffer总结
- 无须require的一个全局变量
- 实现NodeJS平台下的二进制数据操作
- 不占据V8堆内存大小的内存空间
- 内存的使用由Node来控制，有V8的GC来回收
- 一般配合Stream流使用，充当数据缓冲区
