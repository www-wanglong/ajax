## 数据类型
### 数值 number
JavaScript内部所有的数字都是以64位浮点数形式存储，整数也是如此.
NaN不是一个独立的数据类型，而是一个特殊的数值，它的数据类型依然属于Number
+ 字符串 string
+ 布尔值 boolean
+ null
+ undefined
+ 对象 object
+ symbol ->es6
### null和undefined区别
`null`是表示一个'空对象'，转为数值时为0； `undefined`是一个表示'此处无定义'的原始值，转为数值时为NaN
